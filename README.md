# The Crpyto Table 

A crpyto table with automatic sorting and refresh on spot and future market.

## How to open:
```
cd <project location>
npm install
npm start
```

## Tech Stack
* ReactJS
* TailwindCSS
* Fontawesome
* Axios
* React-Router-Dom
